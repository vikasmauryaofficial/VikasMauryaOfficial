# Vikas Maurya — Portfolio Website

A modern, dark-themed portfolio website for Vikas Maurya, Assistant Professor & CS Researcher.

## 🚀 Deploy in 1 Minute

### Option 1: Vercel (Recommended)
1. Create a free account at [vercel.com](https://vercel.com)
2. Install Vercel CLI: `npm i -g vercel`
3. Run `vercel` in this folder and follow prompts
   
   **OR** drag-and-drop this folder at [vercel.com/new](https://vercel.com/new)

### Option 2: Netlify
1. Create a free account at [netlify.com](https://netlify.com)
2. Drag-and-drop this entire folder onto [app.netlify.com](https://app.netlify.com)
   
   **OR** via CLI: `npm i -g netlify-cli` then `netlify deploy --prod --dir .`

### Option 3: GitHub Pages
1. Push this folder to a GitHub repository named `<yourusername>.github.io`
2. Go to repo Settings → Pages → Source: Deploy from branch → `main` → `/ (root)`
3. Your site will be live at `https://<yourusername>.github.io`

## 📁 Files
- `index.html` — The entire portfolio (single file, zero dependencies)
- `vercel.json` — Vercel deployment config
- `netlify.toml` — Netlify deployment config
- `README.md` — This file

## ✨ Features
- Fully responsive (mobile + desktop)
- Custom animated cursor
- Scroll-triggered reveal animations
- Dark aesthetic with accent colors
- All sections: Hero, About, Education, Publications, Projects, Certifications, Awards, Contact
- Zero npm dependencies — pure HTML/CSS/JS
- Google Fonts loaded via CDN

## 📝 Customization
Open `index.html` in any text editor to update content. All data is in plain HTML — no build step needed.
